# 🎒 Lost & Found Management System

A full-stack Laravel-based web application to manage lost and found items in an organized and efficient way.

---

## 🚀 Project Overview

This system helps users report lost items and found items while allowing admins to manage, update, and track all records. It also includes API support for testing with Postman.

---

## ✨ Features

### 👤 User Features
- View all lost & found items
- Search items by name
- Filter items by status (Lost / Found / Returned)
- View dashboard statistics

### 🛠️ Admin Features
- Add new items
- Edit item details
- Delete items
- Import data via Excel
- Export data to Excel & PDF
- Capture system screenshot report

### 🔌 API Features
- REST API for items
- JSON response support
- Postman testing ready

---

## 🧰 Tech Stack

- Laravel 12
- PHP 8+
- MySQL
- Blade Templates
- Bootstrap / Custom CSS
- Chart.js (Dashboard graphs)
- Maatwebsite Excel (Import/Export)
